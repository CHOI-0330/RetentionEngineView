# 일본 건설/토목 업계 B2B 서비스 디자인 조사 리포트

## 📋 Executive Summary

일본 건설/토목 업계 B2B 서비스는 **신뢰감**, **안정감**, **전문성**을 핵심 가치로 하는 보수적이면서도 현대적인 디자인 접근이 특징입니다. 주요 서비스(ANDPAD, Photoruction, 建設BALENA)의 분석 결과, 업계는 점진적으로 디지털 전환(DX)을 진행 중이며, 사용자 경험 개선을 위한 디자인 시스템 구축에 주력하고 있습니다.

## 1. 일본 건설업계 대표 서비스 UI/UX 특징

### 1.1 ANDPAD (アンドパッド)
- **개요**: 이용사 수 20만개, 사용자 수 51만명의 건설 프로젝트 관리 SaaS
- **핵심 특징**:
  - 「Tsukuri」라는 자체 디자인 시스템 운영
  - 2024년 7월 OKLCH 컬러 스페이스 기반 카라ー파레트 전면 개편
  - 디자인토큰 기반의 일관된 컴포넌트 시스템
  - 모바일/태블릿 우선 설계 (현장 중심 UX)

- **디자인 철학**:
  - "ANDPAD 다움"의 일관성 유지
  - 인지 패턴의 체계적 재설계
  - CSS 공통화를 통한 점진적 시스템 구축

**출처**: [ANDPADに学ぶ、デザインシステム導入の"落とし穴"](https://careerhack.en-japan.com/report/detail/1554), [アンドパッドのデザイナーについて紹介](https://tech.andpad.co.jp/entry/2023/06/28/100000)

### 1.2 Photoruction (フォトラクション)
- **개요**: 30만+ 건설 프로젝트 도입, 사진 관리부터 BIM까지 통합 플랫폼
- **핵심 특징**:
  - 건설 BPaaS (SaaS + AI-BPO) 제공
  - 360도 사진 기반 바추얼 투어 기능 (2024.12 리코와 협업)
  - 모바일 앱 중심의 현장 작업 효율화
  - 검사/기록/순회 업무의 직관적 인터페이스

- **디자인 특징**:
  - 현장 중심 UI (스마트폰 최적화)
  - 시각적 정보 중심 (사진/도면 관리)
  - 실시간 협업을 위한 클라우드 기반 UX

**출처**: [建設業界2024年問題の救世主](https://media.dglab.com/2021/08/18-photoruction-01/), [リコー、フォトラクションとともに360度写真を用いた建設業向けサービスを提供](https://blog.ricoh360.com/ja-news/2024-12-10)

### 1.3 建設BALENA (バレーナ)
- **개요**: 중소 건설업 특화 업무 통합 시스템, 월 16,500엔 정액제
- **핵심 특징**:
  - 무제한 커스터마이징 가능
  - 극도로 심플한 조작성 (항목 입력만으로 완료)
  - iOS 단말에서 현장 일보 직접 입력
  - 원격 서포트를 통한 친절한 온보딩

- **디자인 특징**:
  - 업무 중심의 기능적 레이아웃
  - PC 미숙련자도 사용 가능한 단순화된 UI
  - 일본 건설업 29개 업종 각각의 업무 실태 반영

**출처**: [バレーナとは](https://office-concierge.co.jp/balena_about/), [建設BALENAの口コミ評判や価格](https://www.shopowner-support.net/attracting_customers/construction/construction-industry/balena/)

## 2. 색상 팔레트 - 건설/토목 업계 주요 색상

### 2.1 Primary Colors (주조색)

#### **블루 계열** 🔵
- **사용 이유**: 신뢰감, 안정감, 청결감 표현
- **연상**: 하늘, 물, 지구 → 건설업의 기반
- **특징**:
  - 법인 대상 업계와 IT 기업에서 가장 많이 사용
  - 과도하게 사용해도 시각적으로 부담이 적음
  - 보수적인 일본 기업 문화에 적합
- **권장 팔레트**:
  - Primary Blue: `#0066CC` ~ `#003D82` (중후한 기업 블루)
  - Sky Blue: `#4A90E2` ~ `#5AB0E8` (밝고 친근한 블루)
  - Navy: `#1A2B4A` ~ `#0A1929` (전문성/권위)

#### **오렌지 계열** 🟠
- **사용 이유**: 에너지, 활력, 따뜻함 표현
- **특징**:
  - 사용자의 구매 의욕 자극 (CTA 버튼에 효과적)
  - 시선을 끄는 강조 포인트
  - 블루와 대비색 관계로 조화로움
- **권장 팔레트**:
  - Vivid Orange: `#FF6B35` ~ `#FF8C42` (액센트용)
  - Warm Orange: `#FF9F43` ~ `#FFB84D` (친근함)
  - Burnt Orange: `#D67A3A` ~ `#C76A2F` (안정감)

**출처**: [日本 建設業 Webサイト デザイン 配色](https://sankoudesign.com/category/blue/)

### 2.2 Secondary Colors (보조색)

#### **그레이 계열** ⚫⚪
- **사용 목적**: 텍스트, 배경, 구분선
- **권장 팔레트**:
  - Dark Gray: `#2C3E50` ~ `#34495E` (헤딩/본문)
  - Medium Gray: `#7F8C8D` ~ `#95A5A6` (서브 텍스트)
  - Light Gray: `#ECF0F1` ~ `#F8F9FA` (배경)
  - Border Gray: `#DFE6E9` ~ `#E8EAED` (구분선)

#### **그린 계열** 🟢
- **사용 목적**: 성공 메시지, 환경/안전 강조
- **권장 색상**: `#27AE60` ~ `#2ECC71` (상태 표시용)

#### **레드 계열** 🔴
- **사용 목적**: 경고, 에러, 긴급 알림
- **권장 색상**: `#E74C3C` ~ `#C0392B` (신중하게 사용)

### 2.3 컬러 시스템 구축 원칙

ANDPAD의 사례에서 배운 원칙:
1. **명도 차이를 포함한 톤 팔레트** 구성
2. **WCAG 2.0 AA 준수**: 텍스트/배경 대비율 4.5:1 이상
3. **OKLCH 컬러 스페이스** 활용으로 일관된 대비율 유지
4. **디자인 토큰** 시스템으로 색상 관리

## 3. 폰트 스타일 - 일본 비즈니스 환경

### 3.1 웹폰트 권장사항

#### **신뢰감·안심감 중시** (BtoB 최우선)
일본 모리사와(Morisawa)의 BtoB 사이트 권장 폰트:

1. **고딕체 (ゴシック体)**
   - **사용 이유**: 가독성, 현대적, 깔끔함
   - **추천**: Noto Sans JP, Hiragino Kaku Gothic Pro
   - **용도**: UI 요소, 본문, 헤더

2. **명조체 (明朝体)**
   - **사용 이유**: 전통, 권위, 격식
   - **추천**: Noto Serif JP, Hiragino Mincho Pro
   - **용도**: 제목, 기업 소개, 공식 문서

### 3.2 폰트 사이즈 시스템

```css
/* 권장 타이포그래피 스케일 */
--font-size-h1: 32px;  /* 메인 헤딩 */
--font-size-h2: 24px;  /* 섹션 제목 */
--font-size-h3: 20px;  /* 서브 제목 */
--font-size-body: 16px; /* 본문 (일본어 가독성) */
--font-size-small: 14px; /* 보조 텍스트 */
--font-size-caption: 12px; /* 캡션/주석 */

/* 폰트 굵기 */
--font-weight-regular: 400;
--font-weight-medium: 500;
--font-weight-bold: 700;

/* 줄 간격 */
--line-height-tight: 1.2;
--line-height-normal: 1.6; /* 일본어 적정 */
--line-height-loose: 1.8;
```

### 3.3 일본어 특화 고려사항

- **일본어는 22pt 미만 텍스트**: 배경 대비 4.5:1 필수
- **한자/가나 혼용**: 고딕체가 가독성 우수
- **웹에서는 약간 굵은 웨이트** 선택 (이미지 배경에서 가독성)
- **세로쓰기 지원** 고려 (전통적 문서)

## 4. 레이아웃 패턴 - 보수적 일본 기업 사용자

### 4.1 신뢰감을 주는 레이아웃 원칙

#### **1. 명확한 정보 계층 구조**
```
[로고/회사명] - [내비게이션] - [검색/문의]
         ↓
[히어로 섹션: 메인 메시지 + CTA]
         ↓
[서비스 소개: 3-4 카드 그리드]
         ↓
[실적/사례: 타임라인 or 그리드]
         ↓
[고객 신뢰 요소: 로고, 수치, 인증]
         ↓
[CTA 섹션: 문의/자료청구]
         ↓
[푸터: 회사정보/링크/연락처]
```

#### **2. Grid 시스템**
- **12 컬럼 그리드** 표준 사용
- **카드 레이아웃**: 3열(데스크탑), 2열(태블릿), 1열(모바일)
- **여백**: 충분한 화이트스페이스로 안정감 표현

#### **3. 반응형 디자인 필수**
- 2024년 현재 레스포ン시브 미지원 = 신뢰도 하락
- 모바일 우선 접근 (현장 작업자 타깃)

**출처**: [BtoBデザインを開発会社のデザイナーが事例とともに解説](https://techracho.bpsinc.jp/iizuka/2025_03_05/148982)

### 4.2 건설업 특화 레이아웃 요소

#### **실적/사례 페이지**
```
┌─────────────────────────────────────┐
│  [필터링: 업종/규모/연도]              │
├─────────────────────────────────────┤
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │사진 │  │사진 │  │사진 │  │사진 │ │
│  ├─────┤  ├─────┤  ├─────┤  ├─────┤ │
│  │프로젝트명  │        │        │     │
│  │개요/규모   │        │        │     │
│  │시공기간    │        │        │     │
│  └─────┘  └─────┘  └─────┘  └─────┘ │
└─────────────────────────────────────┘
```

#### **서비스 소개**
- **Before/After 비교**: 도입 효과 시각화
- **프로세스 플로우**: 단계별 설명 (1→2→3→4)
- **수치 강조**: 도입사 수, 프로젝트 수, 절감율

#### **신뢰 구축 요소**
- ISO 인증서, 건설업 허가증
- 대형 프로젝트 실적 (로고 진열)
- 안전관리 실적 (무재해 일수 등)
- 고객 후기/인터뷰

**출처**: [建設業のホームページ制作で失敗しない業界特化型デザイン戦略](https://sk-t.com/media/seo/20250430-1814/)

### 4.3 UI 컴포넌트 패턴

#### **버튼 (CTA)**
```css
/* Primary Button (주요 액션) */
background: #FF6B35; /* 오렌지 */
color: #FFFFFF;
padding: 12px 32px;
border-radius: 4px; /* 약간의 둥근 모서리 */
font-weight: 700;

/* Secondary Button (보조 액션) */
background: transparent;
border: 2px solid #0066CC; /* 블루 */
color: #0066CC;
```

#### **카드 컴포넌트**
```css
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
border-radius: 8px;
padding: 24px;
background: #FFFFFF;
```

#### **입력 폼**
- 라벨 명확 (필수/선택 표시)
- 에러 메시지 친절 (빨간색 + 아이콘)
- 자동 완성 지원
- 입력 가이드 툴팁

## 5. 아이콘/일러스트 스타일

### 5.1 아이콘 스타일 가이드

#### **Line Icon (선형 아이콘)**
- **사용 이유**: 깔끔하고 현대적, 프로페셔널
- **특징**:
  - 2px 또는 2.5px 선 굵기
  - 24×24px 기본 사이즈
  - 배경 대비 3:1 이상 대비율 (WCAG)
  - 단색 또는 브랜드 컬러

#### **Filled Icon (채워진 아이콘)**
- **사용 시기**: 선택/활성 상태 표시
- **예시**:
  - 체크 마크 (완료)
  - 별 (즐겨찾기)
  - 홈 (현재 페이지)

### 5.2 일러스트 스타일

#### **건설업 적합 스타일**

**1. Flat Illustration (플랫 일러스트)**
- 단순하고 깔끔한 2D 스타일
- 명확한 컬러 블록
- 아이소메트릭 건물/현장 표현

**2. Isometric Illustration (아이소메트릭)**
- 3D 느낌의 2.5D 표현
- 건설 현장, 장비, 프로세스 시각화
- 전문성과 현대성 동시 표현

**3. Icon-based Infographics**
- 복잡한 프로세스를 아이콘으로 단순화
- 숫자/통계 데이터 시각화
- 플로우차트 형태

#### **피해야 할 스타일**
- ❌ 지나치게 캐주얼한 캐릭터
- ❌ 손그림 느낌 (Handdrawn)
- ❌ 추상적이고 모호한 표현

### 5.3 이미지 사용 가이드

- **실제 현장 사진** 우선 (진정성)
- **Before/After 비교 사진** (효과 입증)
- **장비/기술 클로즈업** (전문성)
- **팀 사진** (신뢰감, 휴먼터치)

## 6. 전반적인 톤앤매너

### 6.1 신뢰감 (Trust)

#### **디자인 전략**
- ✅ **일관된 브랜딩**: 로고, 컬러, 폰트 통일
- ✅ **명확한 정보**: 가격, 조건, 프로세스 투명 공개
- ✅ **실적 강조**: 수치, 로고, 사례 배치
- ✅ **인증/자격**: 눈에 띄는 위치 배치
- ✅ **고객 후기**: 실명/소속/사진 포함

#### **컨텐츠 원칙**
- 과장하지 않는 정직한 표현
- 전문 용어 사용 (+ 설명 툴팁)
- 법적 근거/규정 준수 명시

**출처**: [信頼感を与えるデザインとは？](https://fumufumuui.com/posts/trustworthy-design)

### 6.2 안정감 (Stability)

#### **비주얼 요소**
- **좌우 대칭 레이아웃**
- **수평선 강조** (헤더, 구분선)
- **규칙적인 그리드**
- **충분한 여백** (답답함 방지)

#### **색상 활용**
- 블루 계열 주조색 (안정감)
- 낮은 채도의 배경 (차분함)
- 높은 명도 대비 피하기

### 6.3 전문성 (Professionalism)

#### **타이포그래피**
- 적절한 크기 (가독성)
- 계층 구조 명확
- 전문 용어 정확한 표기

#### **데이터 시각화**
- 차트/그래프로 성과 표현
- 인포그래픽으로 프로세스 설명
- 타임라인으로 연혁 정리

#### **기술 강조**
- **3D/360도 뷰어** (Photoruction 사례)
- **BIM 통합** 표현
- **AI/자동화** 기능 소개

### 6.4 접근성 (Accessibility)

일본 기업은 **감성보다 합리성**을 중시:

- **논리적 정보 구조**
- **검색 기능 강화**
- **FAQ/지원 접근 용이**
- **다운로드 자료** 제공 (PDF 카탈로그)

### 6.5 2024 트렌드 반영

#### **AI 통합 표현**
- 53% 기업이 AI 기술로 제품 혁신 중

## 10. 결론 및 권장사항

### ✅ 핵심 인사이트

1. **신뢰감 최우선**: 일본 건설업계 B2B는 감성보다 신뢰/안정/전문성
2. **블루 + 오렌지**: 안정감(블루)과 활력(오렌지)의 조화
3. **고딕체 중심**: Noto Sans JP로 가독성과 현대성 확보
4. **모바일 최적화**: 현장 작업자를 위한 반응형 필수
5. **실적 중심 컨텐츠**: 수치, 로고, 사례로 신뢰 구축
6. **점진적 DX**: 보수적 문화에 맞춘 단계적 혁신

### 🎯 즉시 적용 가능한 액션

1. **컬러**: Primary Blue (#0066CC) + Secondary Orange (#FF6B35)
2. **폰트**: Noto Sans JP (400/500/700 웨이트)
3. **레이아웃**: 12 컬럼 그리드, 8px 간격 시스템
4. **컴포넌트**: 둥근 모서리(4-8px), 미묘한 그림자
5. **컨텐츠**: 실적 수치, 인증서, 고객 로고 배치
6. **접근성**: WCAG AA 준수 (대비율 4.5:1)
