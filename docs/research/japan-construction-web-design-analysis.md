## 1. 색상 사용 패턴

### 주요 발견사항
일본 건설업계는 **AI가 선호하는 화려한 파랑/보라가 아닌** 실제 신뢰감을 주는 색상을 사용합니다.

#### 구체적 색상 팔레트:

**大林組 (Obayashi)**
- 메인 컬러: **진한 녹색 (다크 그린)** - 브랜드 아이덴티티
- 로고의 그린 삼각형이 강력한 시각적 앵커
- 배경: 깨끗한 화이트
- 액센트: 최소한의 블루 링크

**鹿島建設 (Kajima)**
- 메인 컬러: **빨간색 (레드)** - "100年をつくる会社"
- 로고와 브랜드에서 빨강 적극 사용
- 매우 보수적이고 전통적인 느낌
- 배경: 화이트와 라이트 그레이

**清水建設 (Shimizu)**
- 메인 컬러: **블루 (로얄 블루)** - 회사 로고
- 하지만 AI스러운 "선명한 파랑"이 아닌 **차분한 네이비/로얄 블루**
- 배경: 대부분 화이트
- "子どもたちに誇れるしごとを。" 캐치프레이즈에 큰 비중

**大成建設 (Taisei)**
- 메인 컬러: **딥 블루 (네이비 계열)**
- 실제 현장 사진의 자연색이 주를 이룸 (흙색, 하늘색, 녹색)
- 최소한의 컬러 사용, 사진이 주인공
- 액센트: 밝은 블루 버튼

**竹中工務店 (Takenaka)**
- 메인 컬러: **차분한 톤의 혼합**
- 실제 건축물 이미지의 자연스러운 색상
- 브랜드 색상보다는 **콘텐츠 중심**
- 화이트 배경 + 고급스러운 세리프 폰트

### 🔑 핵심 포인트
- **AI스러운 선명한 파랑/보라색 거의 없음**
- **회사별 전통적인 코퍼레이트 컬러 유지** (녹색, 빨강, 블루 등)
- **자연스러운 사진의 색상이 전체 톤을 지배**
- 배경은 거의 항상 **화이트**
- 그라데이션 거의 없음

## 2. 레이아웃 구조

### Hero 섹션 vs 정보 중심

**공통점: 모두 대형 Hero 이미지 사용**
- 하지만 "디자인을 위한 디자인"이 아님
- 실제 시공 현장이나 완공된 건물 사진

#### 구조 비교:

**大林組**
```
Hero: 풀스크린 슬라이드 (실제 프로젝트 사진)
  → 프로젝트명 + "詳細を見る" 버튼

아래:
- 긴급 공지 (노란색 배너)
- 뉴스 리스트 (시간순 배치)
- 특집 프로젝트 카드 그리드
- SNS 링크
```

**鹿島建設**
```
Hero: "100年をつくる会社" 대형 타이포그래피
  → 실제 공사 현장 사진 배경

아래:
- 탭 구분 뉴스 (プレスリリース, IRニュース, 更新情報)
- 픽업 콘텐츠 (큰 이미지 카드)
- 사업 영역 아이콘 그리드
```

**清水建設**
```
Hero: 실제 현장 사진 풀스크린
  → "子どもたちに誇れるしごとを。" 캐치프레이즈

아래:
- 경고 배너 (詐欺 주의)
- 최신 뉴스 리스트 (날짜 + 카테고리)
- "おすすめコンテンツ" 슬라이더
- 사업 토픽스 큰 카드
```

**大成建設**
```
Hero: 실제 공사 현장 영상/슬라이더
  → 프로젝트 설명 오버레이

아래:
- NEWS 탭 시스템 (すべて, リリース, お知らせ, IR 등)
- TOPICS 슬라이더
- "地図に残る仕事。®" 특집 섹션
```

**竹中工務店**
```
Hero: 세련된 건축 작품 슬라이더
  → 깨끗한 타이포그래피

아래:
- News (시간순 리스트)
- Pick UP (테마별 큰 카드)
- People & Story (인터뷰)
- 건축 작품 갤러리
```

### 🔑 핵심 포인트
- **Hero는 화려함이 아닌 "실제성"**
- **정보 밀도가 높음** (한 화면에 많은 콘텐츠)
- **수직 스크롤 긴 페이지**
- 탭/카테고리로 정보 정리
- 버튼은 최소한 ("詳細を見る", "一覧を見る")

## 3. 타이포그래피 스타일

### 폰트 선택

**일본어 웹사이트의 특징:**
- 고딕체(산세리프)가 주류
- 하지만 **단순한 고딕이 아닌 가독성 높은 폰트**

#### 실제 사용 예:

**大林組**
- 메인: 고딕체 (標準ゴシック 계열)
- Hero 타이틀: 굵은 고딕 (明朝체 아님)
- 본문: 중간 굵기 고딕
- **특징: 깨끗하고 현대적, 하지만 과하지 않음**

**鹿島建設**
- 강렬한 고딕체 사용
- "100年をつくる会社" - 매우 굵은 폰트
- 전통과 신뢰를 강조하는 웨이트
- **특징: 힘있고 보수적**

**清水建設**
- 균형잡힌 고딕체
- "子どもたちに誇れるしごとを。" - 부드러운 굵기
- 본문은 표준 굵기
- **특징: 친근하면서도 전문적**

**大成建設**
- 깔끔한 고딕체
- "For a Lively World" - 얇은 세리프 (영문)
- 정보 밀도 높은 폰트 사용
- **특징: 모던하고 정돈됨**

**竹中工務店**
- 세련된 고딕 + 일부 명조체
- 타이틀에 명조체 사용으로 고급스러움
- "想いをかたちに 未来へつなぐ" - 세리프
- **특징: 가장 디자인 지향적**

### 폰트 크기 계층

```
일반적인 패턴:
H1 (Hero): 48-72px (대형)
H2 (섹션): 32-40px
H3 (카드 제목): 20-24px
본문: 14-16px
작은 텍스트 (날짜 등): 12-13px
```

### 🔑 핵심 포인트
- **트렌디한 폰트가 아닌 가독성 우선**
- 폰트 웨이트로 계층 구분 (900, 700, 400, 300)
- 한자가 많아 **행간(line-height)이 넉넉함** (1.6-1.8)
- 과도한 폰트 믹스 없음 (최대 2-3개 폰트 패밀리)

## 4. 이미지 사용 방식

### 이미지 퀄리티와 스타일

**절대 법칙: "실제 사진만 사용"**

#### 구체적 분석:

**大林組**
- 실제 완공 건물 사진 (고해상도)
- 공사 현장 사진 (진행 중인 모습)
- **특징: 있는 그대로, 필터 최소화**
- 이미지 위 흰색 텍스트 오버레이

**鹿島建設**
- 매우 다양한 프로젝트 사진
- 인프라 사진 (다리, 댐, 터널)
- 건물 외관 + 현장 작업 모습
- **특징: 스케일감을 보여주는 구도**

**清水建設**
- 실제 프로젝트의 고품질 사진
- 현장감 있는 각도와 조명
- **특징: 자연광 사진, 보정 최소**
- "子どもたちに誇れる" 메시지와 연결

**大成建設**
- 공사 진행 중 현장 사진 (중장비 포함)
- 완공 후 건물 사진
- 항공 촬영 (대규모 프로젝트)
- **특징: 작업 과정 강조, 땅과 기계**

**竹中工務店**
- 고품질 건축 사진 (예술적 구도)
- 디테일 클로즈업
- 내부 공간 사진
- **특징: 가장 "디자인적", 하지만 여전히 실제 사진**

### 이미지 비율과 레이아웃

```
Hero 이미지: 16:9 또는 21:9 (풀스크린 와이드)
카드 이미지: 4:3 또는 3:2
프로젝트 갤러리: 정사각형 또는 16:9
```

### 이미지 처리 방식

**절대 하지 않는 것:**
- ❌ 일러스트레이션
- ❌ 3D 렌더링이 주가 되지 않음
- ❌ AI 생성 이미지
- ❌ 스톡 사진 느낌
- ❌ 과도한 색보정

**항상 하는 것:**
- ✅ 실제 프로젝트 사진
- ✅ 자연스러운 색감
- ✅ 현장감
- ✅ 고해상도
- ✅ 프로젝트명 명시

### 🔑 핵심 포인트
- **"찍힌" 사진이지 "만든" 이미지가 아님**
- 실제 프로젝트의 증거
- 중장비, 작업자, 진행 과정도 당당히 노출
- 필터나 효과 최소화
- **신뢰감 = 실제성**

## 6. AI스럽지 않은 "진짜" 기업 웹사이트의 특징

### ❌ AI가 만들면 이렇게 하지만, 실제는 다름:

| AI 스타일 | 실제 일본 건설사 스타일 |
|-----------|------------------------|
| 화려한 그라데이션 배경 | 깨끗한 화이트 배경 |
| 일러스트와 아이콘 중심 | 실제 사진 중심 |
| 트렌디한 폰트 믹스 | 고딕체 위주 일관성 |
| 동적 애니메이션 많음 | 최소한의 슬라이드만 |
| "혁신", "미래", "AI" 강조 | 실적과 역사 강조 |
| 감각적인 카피 | 직설적이고 정직한 메시지 |
| 모바일 퍼스트 디자인 | PC와 모바일 균형 |
| 밝고 경쾌한 톤 | 무겁고 진지한 톤 |

### ✅ 진짜 기업 웹사이트의 특징:

1. **정보 밀도가 높음**
   - 한 화면에 많은 콘텐츠
   - 스크롤이 길어도 괜찮음
   - 중요한 정보는 반복 노출

2. **법적 의무사항 철저**
   - 個人情報保護について
   - 電子公告
   - コンプライアンス相談窓口
   - 協力会社の皆様へ

3. **다층적 내비게이션**
   - 메가 메뉴 구조
   - 카테고리가 매우 세분화
   - 모든 페이지로 2-3클릭 이내 접근

4. **뉴스와 업데이트가 중심**
   - 최신성이 신뢰의 지표
   - 날짜가 모든 콘텐츠에 표시
   - 과거 뉴스도 아카이브

5. **실제 사용자 고려**
   - 협력사, 주주, 구직자 등 명확한 타겟
   - 각 타겟별 전용 섹션
   - 실용적인 정보 (채용, 입찰, 공지)

6. **브랜드보다 콘텐츠**
   - 프로젝트가 주인공
   - 회사 로고는 상단에만
   - 자기 자랑이 아닌 실적 증명

7. **보수적이지만 디지털**
   - 최신 웹 기술 사용 (반응형, 슬라이더)
   - 하지만 검증된 디자인 패턴
   - 실험적 UI 거의 없음

## 7. 한국 건설업체 적용 시 권장사항

### DO (해야 할 것):

1. **실제 프로젝트 사진 중심**
   - 현장 사진, 완공 사진 고해상도로
   - 시공 과정도 당당히 보여주기
   - 중장비와 작업자 모습 포함

2. **명확한 코퍼레이트 컬러 유지**
   - 트렌드보다 브랜드 아이덴티티
   - 일관된 색상 시스템
   - 과도한 색상 사용 자제

3. **정보 중심 레이아웃**
   - 뉴스와 공지사항 최상단
   - 날짜 명확히 표시
   - 카테고리 분류 체계적

4. **신뢰감 주는 요소 강화**
   - 수상 내역
   - 대형 프로젝트 실적
   - 기업 역사와 전통
   - 법적 고지사항

5. **가독성 높은 타이포그래피**
   - 단순 명료한 폰트
   - 충분한 행간
   - 명확한 계층 구조

### DON'T (피해야 할 것):

1. **트렌디한 디자인 요소**
   - 그라데이션 배경
   - 화려한 애니메이션
   - 추상적인 일러스트
   - AI 생성 이미지

2. **모호한 메시지**
   - 감성적이기만 한 카피
   - 구체성 없는 비전
   - 과장된 표현

3. **정보 숨기기**
   - 미니멀리즘의 과도한 적용
   - 클릭을 유도하는 티저
   - 중요 정보를 하위 페이지에

4. **일관성 없는 디자인**
   - 페이지마다 다른 스타일
   - 섹션별 색상 통일 안됨
   - 이미지 비율 제각각

### 3가지 키워드:

1. **실제성 (Authenticity)**

   - 법적 책임감
   - 역사와 전통

3. **전문성 (Professionalism)**
   - 기술력 강조
   - 데이터 기반
   - 체계적 구조
